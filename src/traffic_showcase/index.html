Traffic Optimisation Model Showcase

Project Overview & Architecture

This project demonstrates the application of various machine learning models for traffic volume prediction. The goal is to analyze and compare the performance of models like Linear Regression, Random Forest, and XGBoost in forecasting traffic conditions. Key features used for prediction include time of day, day of the week, month, weather conditions, and junction-specific data.
